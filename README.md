### What's the whole point of this site?
    Essentially, it's just a way to practice your JS, HTML, and CSS skills. The site has (or will have) several useful features. Currently, it's a currency converter.

The site is styled in Windows 10 Fluent Design (with an emphasis on the style from the Settings app). The following features, taken from the original Windows, are present:
- Style
- Taskbar
- Site icon on the desktop
- Notification Center (but it displays a list of changes)
- Activation keys taken from the Microsoft website
- A pack of wallpapers, cursors, etc.
- Dark/light theme that switches automatically
- Currency convetrter

The idea came to me by chance, and I plan to develop it further.
